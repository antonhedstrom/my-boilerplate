Endpoints

Method Path Description Notes
GET /book Get a list of Books. Returns array of <Book>.
  • limit (optional) (default 10) - Max number of records.
GET /book/:id Get a single Book. Returns a <Book>.
GET /user Get a list of Users. Returns array of <User>.
  • limit (optional) (default 10) - Max number of records.
GET /user/:id Get a single User. Returns a <User>.
